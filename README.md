# IsQRSecure

A free QR code safety scanner for iOS, built to help everyday users check whether a QR code is safe before visiting the link behind it.

## About

IsQRSecure was created by Jake Ives, a security professional, after seeing users regularly fall for QR phishing (quishing) scams. The app walks you through five quick awareness questions and gives you a clear safety verdict before you visit any scanned link.

No data is collected. No accounts required. No ads. Completely free.
